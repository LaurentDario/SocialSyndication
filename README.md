SocialSyndication
=================

Wordpress Plugin - Fetching social media feeds
Plugin en développement

Ce plugin vous permet de fetcher les réseaux supportés à la recherche de mots clés.
Chaque élément de flux est enregistré comme un custom-post dans le blog pour permettre l'accès à toutes les fonctionnalités de WordPress.

Pour installer, placez le dossier social-syndication sous wp-content/plugins
